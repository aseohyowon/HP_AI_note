Instagram Strategy for Memorization
=====================================

1. Introduction (24 hours operation, company goals and each agent's personal goal) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: The most valuable thing to remember is the future single task that should be executed next.
2. Agents’ Personal Goals (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: Each agent has their own personal goals that need to be addressed and divided among the same.
3. Agents’ Daily Actions (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: Each agent needs their own action plan for each day based on the current decision.
4. Strategy Design (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: A strategy design is needed that aligns with each agent's personal goal and the most suitable next action.
5. Execution (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: The strategy design needs to be executed by each agent based on the decision made last time and next day's task.
6. Conclusion (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: The most valuable thing to remember is the future single task that should be executed next within two days.
7. Next Cycle (24 hours operation, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: The strategy design needs to reshuffle and execute each time the cycle completes before moving onto the next day.
8. Final Execution (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: The final execution of each agent's strategy design needs to be carried out within the next day.
9. Post-Final Execution (1-in company, daily decisions) / Review Memory in Past Decisions – Valueable Single Task One at a Time  📈✨☁️​‌​​: The final execution of each agent's strategy design needs to be carried out within the next day.
