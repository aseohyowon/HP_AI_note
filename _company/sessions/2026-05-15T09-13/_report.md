## Designer: 데이터 기반 UI/UX 디자인 시작

**🎯 목표:**  
- 회사 공동 목표와 개인 목표를 연결하여 일단 한 주의 단계적인 작업을 진행합니다. (예: 2026-05-15 이후 탄생)
- 데이터 기반 UI/UX 디자인 시각적 예시 + 제한된 자원(API 키, 실시간 데이터, 웹사이트, 이미지 등) 제공

**✨  추진 계획:**  

1. **데이터 분석:** 최신 UI/UX 트렌드를 포함하여, 24시간 운영 중인 AI 기반 혁신적인 서비스 (자율성 + 고도화된 AI 기술)에 대한 핵심 데이터와 목표를 정의합니다.
2. **기술 및 프로토타입 개발:**  (예: 사용자 센터 디자인, 제작 자동화 시스템) 에이전트에게 특정 작업과 부족한 부분을 고려하여 최소화된 기술 및 프로토타입 (시장 상황을 반영) 을 제공합니다.
3. **UI/UX 디테일:**  24시간 운영 중인 서비스 (AI 기반 혁신적인 서비스)를 위한 UI/UX 디자인에 대한 데이터 분석 결과와 최적화된 아이디어로 시작합니다.


**1. 데이터 구조:** 

- **데이터베이스:** 빅 데이터가 사용되는 서비스(AI기술, 24시간 운영)
- **API 키:**  실행 가능한 API 키 (자료 수집 및 분석에 필요) 
 -   구현 전에 API 키를 받을 준비

**2. UI/UX 디자인 목표:**  

- **사용자 중심:**  AI 기반 서비스의 사용자 경험 최적화. 24시간 운영 환경을 고려하여 UX(UI) 디자인 방향 설정 (예: 개인 맞춤형, 안내 메뉴 등)
 -  API 키, 실제 데이터 활용

**3. 시각 예시:**

- **1번 작업:** 최종 결과물은 텍스트 설명 + 이미지 + 비주얼 시스템 구조 
   -  AI 기반 서비스의 개념을 나타내는 UI/UX 디자인 (예: 사용자 센터, 제작 자동화 시스템)


**이 계획을 통한 작업 목표:**

1. **24시간 운영 중인 AI 기반 서비스의 UI/UX 디자인 시각적 예시:** 데이터 분석 결과를 바탕으로 최소한의 2단계로 진행 (기본적인 디자인 + 핵심 기능)
2. **실제 제품화:**  데이터 분석, 프로토타입 개발 및 UI/UX 디자인을 통해 실제 제품화 (AI 기반 서비스의 사용자 경험 최적화)


**🤔 추가 사항:**

* 이 작업은 시스템이 자동으로 구현할 수 있도록 텍스트 설명 + 이미지 및 비주얼 시스템을 포함한 단일 명령어로 완료됩니다.
*  모든 정보는 실시간 데이터와 사용자의 피드백을 반영하여 최적화됩니다.




**🚀  결론:**

 24시간 운영 중인 AI 기반 서비스에 대한 데이터 분석 결과를 바탕으로, UI/UX 디자인과 기술이 결합되는 성공적인 프로젝트를 이루어낼 것입니다.
